# Job & Internship Tracker (API + SQL)

## Overview
This project is a Python-based job and internship tracking system that integrates a public jobs API with a SQL database. It fetches real-world job data, stores it locally, and allows querying and analysis of opportunities.

## Features
- Fetches job/internship listings using a REST API
- Stores data in a SQLite database
- Query and analyze opportunities using Python
- Clean, modular backend design

## Tech Stack
- Python
- REST APIs
- SQLite (SQL)
- Requests library

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the tracker:
   python tracker.py

- Python backend development
